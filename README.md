# pokedex

## Tecnologias Utilizadas: 
### Core
* **React**: Construção da interface via Functional Components e Hooks.
* **JavaScript**: Utilização de recursos modernos como Async/Await, Destructuring e Map/Reduce.
* **CSS**: Estilização com Flexbox, Grid e Variáveis CSS (CSS Variables) para temas dinâmicos.

### Gerenciamento de Dados & Estado
* **TanStack Query (React Query)**: Para requisições assíncronas, cache de dados, tratamento de estados de carregamento e revalidação.
* **Context API**: Para gerenciamento de estado global da UI (controle de Modais).
* **Fetch API**: Para comunicação nativa com a [PokeAPI](https://pokeapi.co/).

### UI & Acessibilidade
* **Radix UI**: Componentes "headless" para garantir a acessibilidade e controle de foco no Modal de detalhes.
* **React Suspense**: Para orquestração de carregamento (Loading Skeletons/Spinners).
